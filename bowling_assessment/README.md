# Bowling Assessment Project

## Files included
- `bowling_game.py` -> main fixed bowling scoring program
- `test_bowling_game.py` -> unit tests
- `example_usage.py` -> simple example run
- `run_tests.py` -> run all tests
- `generate_docs.py` -> generate Python documentation
- `requirements.txt` -> package list
- `.gitignore` -> Git ignore rules

## How to run tests
```bash
python -m unittest -v
```

or

```bash
python run_tests.py
```

## How to generate documentation
```bash
pip install -r requirements.txt
python generate_docs.py
```
